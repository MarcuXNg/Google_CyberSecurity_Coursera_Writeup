### Tools

- Log: record events that occur within an organization's systems
- Splunk: data analysis platform, retain, analyze, search organization's log data
- Chronicle: Cloud-native SEIM tool stores
- Playbooks: a manual that provide details about any operational action
- Network protocal analyzers: a tool designed to capture and analyze data traffic within a network
- SIEM Dashboard: a tool to communicate information or data, organize data into categories and allow analysts to identify potential security incidents, such breaches, as they happen.
